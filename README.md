# Job Application Automation (CLI)

A Python command-line tool that automates resume tailoring by analyzing job descriptions and selecting the most relevant experience from a structured master resume.

---

## Features

- Job description keyword extraction
- Role detection (Cybersecurity, Network, IT)
- Resume bullet selection based on tagged skills
- DOCX resume generation
- Cover letter generation
- Application tracking using CSV
- Windows run.bat execution

---

## Tech Stack

- Python
- Regex
- JSON
- python-docx
- CSV
